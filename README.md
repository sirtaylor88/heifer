## Node Version Manager (nvm)

### Installation steps

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.2/install.sh | bash
```

Create `.nvmrc` file with node version in it.

```bash
echo "21.7.0" >> .nvmrc
```

### Start node

```bash
nvm use
```

## Build

```bash
npm run build
```

## Start

```bash
npm run start
```
